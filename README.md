# Project: 💡*CodeLearn*

### **Давай учиться программированию вместе!** 🚀 *CodeLearn* - твой надежный проводник в мире высоких технологий.

### Описание
Десктопное приложение для изучения терминов по программированию с использованием карточек и системы прогресса.

### 🌌 "Режим обучения"
#### Три основные кнопки:
- «Показать определение» — открывает правильный ответ
- «Знаю» — отмечает термин как изученный и показывает следующий
- «Не знаю» — добавляет термин в список для повторения и показывает следующий
#### Прогресс-бар - показывает твое развитие
#### Панель "?" - показывает случайный термин

### 📊 Основной функционал для пользователя

✅ Возможность отслеживать прогресс по знаниям it терминов<br>
✅ База знаний с определениями<br>
✅ Простой и понятный интерфейс<br>
✅ Быстрый запуск<br>
✅ Подробный просмотр определений

## 🚀 Установка
1. Клонируйте репозиторий:
```bash
git clone <название-репозитория>
cd <название-репозитория>
```

2. Создать и активировать виртуальную среду:
**Windows:**
```bash
python -m venv venv
source venv/Scripts/activate
```
**Mac/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

3. Установить зависимости из файла requirements.txt:
```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

4. Запустить проект:
```bash
python main.py
```
